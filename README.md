# 🌟 Advice App

A premium, modern web application that provides instant wisdom and daily motivation at the click of a button. Built with a focus on high-end aesthetics, smooth transitions, and a seamless user experience.

![Preview Placeholder](https://via.placeholder.com/1200x600/000000/00F2FF?text=Advice+App+Premium+UI)

## ✨ Features

- **Instant Wisdom**: Fetches random pieces of advice from the Advice Slip API.
- **Premium Design**: Crafted with a Material Design 3 inspired aesthetic, featuring glassmorphism and smooth micro-animations.
- **Dynamic Interactions**: Hover effects, glowing buttons, and responsive layouts that feel "alive".
- **Responsive**: Fully optimized for mobile, tablet, and desktop screens.
- **Material Symbols**: Integrated with Material Symbols for clear, modern iconography.

## 🚀 Tech Stack

- **Core**: [React 19](https://react.dev/)
- **Build Tool**: [Vite 8](https://vitejs.dev/)
- **Styling**: [Tailwind CSS 4](https://tailwindcss.com/)
- **Networking**: [Axios](https://axios-http.com/)
- **Icons**: Material Symbols Outlined
- **Typography**: Inter (via Google Fonts)

## 🛠️ Getting Started

### Prerequisites

- **Node.js** (v18.0.0 or higher)
- **pnpm** (Recommended) or npm/yarn

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/Ojas-1008/Advice-App.git
   cd Advice-App
   ```

2. Install dependencies:
   ```bash
   pnpm install
   ```

3. Start the development server:
   ```bash
   pnpm run dev
   ```

4. Build for production:
   ```bash
   pnpm run build
   ```

## 🌐 API Reference

This project utilizes the **[Advice Slip JSON API](https://api.adviceslip.com/)**.
- **Endpoint**: `https://api.adviceslip.com/advice`
- **Method**: `GET`

## 🎨 Design System

The application uses a custom-curated color palette and spacing system:
- **Surface**: Dark, glassmorphic backgrounds.
- **Primary**: Cyan-based glow effects for high-contrast visibility.
- **Typography**: `Inter` for maximum readability across all devices.
